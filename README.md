## Como se usa

Para clonar esta repositorio, vas a necesitar [Git](https://git-scm.com) y [Rust](https://www.rust-lang.org/es)

```bash
# Clone el repositorio.
$ git clone https://github.com/matiasdante/rust-calc.git

# Ir al repo
$ cd rust-passgen

# Construir  
$ cargo build

# Iniciar la app
$ cargo run
```
